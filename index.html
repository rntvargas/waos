🔬 ¿Qué es la Estadística Computacional?

La estadística computacional utiliza métodos computacionales para resolver problemas estadísticos. Incluye simulaciones, inferencia, modelamiento y análisis de datos con herramientas como R, Python y Monte Carlo.
